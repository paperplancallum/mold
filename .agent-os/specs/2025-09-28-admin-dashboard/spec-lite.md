# Admin Dashboard - Lite Summary

Create an admin dashboard at `/admin` that allows the site administrator (callum@paperplan.co) to monitor all customer test submissions across the platform. The dashboard features a sortable/filterable data table, click-through access to full test details, and CSV export functionality for offline analysis.

## Key Points
- Centralized view of all customer test submissions with sorting and filtering
- Admin-only access restricted to authorized email address
- CSV export capability for data analysis and reporting
